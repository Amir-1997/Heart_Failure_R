This project is a personal project that uses UCI-dataset to predict disease outcomes. It has high educational value:

The work includes
 - A full detailed report (in pdf, html and quarto markdown form) on:
       - Data Extraction
       - Exploration
       - Visualization
       - a detailed explanation of the methods and techniques used to build a predictive model
       - Cross-validation reports
       - A plumber api to change the model into a practical program.
